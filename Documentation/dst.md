LA:设计名         # 设计名称 (LA = Label)
ST:针迹数         # 总针迹数 (ST = Stitch Count)
CO:颜色数         # 颜色变化次数 (CO = Color Changes)

# 边界框（相对坐标，0.1mm单位）
+X:最大X坐标      # 正X方向最大偏移
-X:最小X坐标      # 负X方向最小偏移
+Y:最大Y坐标      # 正Y方向最大偏移
-Y:最小Y坐标      # 负Y方向最小偏移

# 起始坐标
AX:起始X坐标      # AX = Anchor X
AY:起始Y坐标      # AY = Anchor Y

# 最大坐标（通常与+X/+Y相同）
MX:最大X          # MX = Maximum X
MY:最大Y          # MY = Maximum Y

# EMBIRD 特有字段
PD:针迹密度       # PD = Point Density
PC:针迹补偿       # PC = Point Compensation
DISTCH:轮廓距离   # Distance for Chevron
DISTCW:线宽距离   # Distance for Column Width
DISTW:走线宽度    # Walking Distance
TSQ:线程质量      # Thread Sequence Quality
